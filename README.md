# Ken's AI Daily 🤖

Daily bilingual AI news digest — English + 繁體中文 — auto-generated by Claude Cowork and published via GitHub Pages.

**Live site:** https://kenchankh97.github.io/ai-daily

## How it works
Each morning, a scheduled Claude Cowork task:
1. Searches for the day's top AI news
2. Drafts posts in English + Traditional Chinese (with inline English terms)
3. Generates an SVG infographic
4. Updates `index.html` with the new post
5. Commits and pushes — GitHub Pages publishes automatically

## Stack
- **Generator:** Claude Cowork (scheduled task)
- **Hosting:** GitHub Pages (free)
- **Format:** Single-file static HTML — no build step, no dependencies

## Folder structure
```
/
├── index.html          ← the website (updated daily)
├── README.md
└── posts/
    └── YYYY-MM-DD/
        ├── post1.md
        ├── post2.md
        └── *.svg
```
